> `<c-s> == Ctrl+s` `<cr>表示回车`

| key                | commond                            | describe                 |
| ----------------   | ---------------                    | ---------------          |
| `<leader>e       ` | `:Ex<cr>                         ` | 打开资源管理器           |
| `<c-s>           ` | `:w<cr>                          ` | 保存                     |
| `Q               ` | `:q!<cr>                         ` | 不保存退出               |
| `<c-q>           ` | `:q<cr>                          ` | 退出如果未保存会提示保存 |

| `<c-l>           ` | `<c-w>l                          ` | 光标移动到右边窗口       |
| `<c-up>          ` | `:res +2<cr>                     ` | 调整窗口大小             |
| `<c-down>        ` | `:res -2<cr>                     ` | 调整窗口大小             |
